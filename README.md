# Bloglist Application

An application to save list of blogs. Live view at https://boiling-savannah-26849.herokuapp.com/

Features:
- Log in/Log out/Sign up
- Create new blog
- Add comment
- Vote for a blog
- Show blogs created by specified user

Tech:
- MongoDB
- Express
- jsonwebtoken
- React
- Redux

How to use:
```bash
npm install
npm run build
npm start
```
